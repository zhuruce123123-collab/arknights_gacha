# 方舟抽卡模拟

明日方舟抽卡模拟器，支持单抽/十连/卡池切换/签到/背包管理。

## 命令

| 命令 | 说明 |
|------|------|
| `/方舟抽卡` | 单抽 (消耗 600 合成玉) |
| `/方舟十连` | 十连抽 (消耗 6000 合成玉) |
| `/方舟卡池` | 查看当前可用卡池 |
| `/方舟切换卡池 <ID>` | 切换当前卡池 |
| `/方舟背包` | 查看干员背包 |
| `/方舟签到` | 每日签到 (获得 200 合成玉) |
| `/方舟抽卡统计` | 查看抽卡统计与保底进度 |
| `/方舟更新数据` | 从 GitHub 更新卡池数据 |

## 配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `data_source` | `github` | 数据来源: `github` / `gitee` |
| `daily_orundum` | `200` | 每日签到合成玉数量 |
| `single_pull_cost` | `600` | 单抽消耗合成玉 |
| `ten_pull_cost` | `6000` | 十连消耗合成玉 |
| `base_rate_6star` | `2` | 6星基础概率(%) |
| `base_rate_5star` | `8` | 5星基础概率(%) |
| `soft_pity_start` | `40` | 软保底起始抽数 |
| `hard_pity` | `50` | 硬保底抽数 |

## 抽卡规则

- 6星基础概率 2%，5星基础概率 8%
- 软保底: 第 40 抽起每抽 +2% 6星概率
- 硬保底: 第 50 抽必出 6星
- 十连保底: 至少 1 个 5星及以上
- 硬核模式: 初始 0 合成玉，需每日签到获取
