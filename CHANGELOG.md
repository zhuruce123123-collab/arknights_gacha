# 更新日志

## [2.4.1] - 2026-06-13

### 修复
- 修复 AstrBot 未传递 config 参数导致插件加载失败，config 改为可选参数

## [2.4.0] - 2026-06-13

### 修复
- 修复 MaterialDataLoader 缺少 config 参数导致插件加载失败
- 统一常量到 constants.py，消除 banner.py / crafting.py / bootstrap_data.py 中的重复定义
- 新增素材下载镜像代理回退机制（edgeone / ghfast / gh-proxy）
- 新增 UP 干员名称匹配和限定干员识别（基于 itemObtainApproach）
- 抽卡引擎重构：RNG 与 DB 操作分离，支持 cost 扣费参数
- SQL 列名白名单防止注入
- 素材更新间隔可配置（auto_update_days）

## [2.3.0] - 2026-06-13

### 修复
- 所有命令添加 event.stop_event()，阻止 LLM 在插件命令执行后继续回复

## [2.2.0] - 2026-06-13

### 新增
- 游戏素材渲染器（AssetGachaRenderer），完美还原明日方舟抽卡界面
- 十连结果使用游戏原版 1280x720 模板合成（rarity 底图 + 干员肖像拼接）
- 单抽结果使用游戏原版角色立绘（1280x720）
- 素材缺失时自动 fallback 到程序化卡片渲染，确保所有干员都能正常显示
- 新增 `resource/gacha_assets/` 素材目录（rarity 模板 + 十连肖像 + 单抽立绘）

### 修复
- 所有命令添加 event.stop_event()，阻止 LLM 在插件命令执行后继续回复
- 修复十连保底数据不一致问题（单事务重写 pull_ten）
- 修复扣费与抽卡非原子操作问题
- 实现 UP 干员名称匹配机制
- 修复限定干员识别（基于 itemObtainApproach）
- 修复临时文件泄漏（上下文管理器）
- 修复素材下载镜像代理回退
- 修复多个防御性编程问题（除零、bare except、字段名错误）
- 统一代码常量，消除重复代码

## [2.1.0] - 2026-06-13

### 重构

- 优化素材合成树渲染，支持动态高度计算
- 优化掉落关卡列表，添加交替背景色

### 新增
- 稀有度渐变背景效果（6星红橙/5星金黄/4星淡紫/3星蓝灰）

- 干员头像加载支持（resource/avatars 目录）
- NEW 徽章标记新干员
- 重复次数标记（x2, x3...）
- 十连结果底部统计信息（各星级数量 + 概率说明）

## [2.0.0] - 2026-06-12

### 变更
- 合并抽卡模拟和素材合成功能为「明日方舟工具箱」
- 合并 `/方舟更新数据` 命令，一次更新卡池和素材数据
- 修复素材查询命令的参数解析

### 新增
- `/方舟素材` — 查看素材合成路线
- `/方舟关卡` — 查看素材掉落关卡
- `/方舟合成` — 查看完整合成成本

- 数据库文件保持不变 (gacha.db, material.db)
- 首次使用请执行 `/方舟更新数据` 更新素材数据
