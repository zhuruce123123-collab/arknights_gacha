# 更新日志

## [2.1.0] - 2026-06-13

### 重构
- 重构抽卡结果 UI 渲染，采用卡片式网格布局
- 优化素材合成树渲染，支持动态高度计算
- 优化掉落关卡列表，添加交替背景色

### 新增
- 稀有度渐变背景效果（6星红橙/5星金黄/4星淡紫/3星蓝灰）
- 五角星图标替代文字星级，避免字体乱码
- 干员头像加载支持（resource/avatars 目录）
- NEW 徽章标记新干员
- 重复次数标记（x2, x3...）
- 十连结果底部统计信息（各星级数量 + 概率说明）

## [2.0.0] - 2026-06-12

### 变更
- 合并抽卡模拟和素材合成功能为「明日方舟工具箱」
- 合并 `/方舟更新数据` 命令，一次更新卡池和素材数据
- 修复素材查询命令的参数解析

### 新增
- `/方舟素材` — 查看素材合成路线
- `/方舟关卡` — 查看素材掉落关卡
- `/方舟合成` — 查看完整合成成本

### 注意
- 数据库文件保持不变 (gacha.db, material.db)
- 首次使用请执行 `/方舟更新数据` 更新素材数据

## [1.0.1] - 2026-06-12

### 修复
- 修复 `_get_user_id` 方法使用了不存在的属性，导致用户 ID 始终为 "unknown" 的问题
- 修正 `metadata.yaml` 中的仓库地址
- 修复抽卡时卡池不存在的问题，自动回退到活跃卡池
- 修复 `_get_arg` 无法解析 `<102>` 格式的参数

## [1.0.0] - 2026-06-11

### 新增
- 单抽 (`/方舟抽卡`)
- 十连抽 (`/方舟十连`)
- 卡池查看与切换 (`/方舟卡池`, `/方舟切换卡池`)
- 每日签到 (`/方舟签到`)
- 背包管理 (`/方舟背包`)
- 抽卡统计 (`/方舟抽卡统计`)
- 卡池数据同步 (`/方舟更新数据`)
- 保底机制 (软保底/硬保底/十连保底)
- 抽卡结果图片渲染
